# API Testing Portfolio

This repository contains examples of API testing performed using Postman.

## Skills Demonstrated

- REST API Testing
- Request & Response Validation
- Status Code Verification
- JSON Validation
- Authentication
- Negative Testing
- Error Handling
- Test Case Design
- Bug Reporting

## Repository Structure

- collections – Postman Collections
- environments – Environment files
- test-cases – Sample API test cases
- bug-reports – Example bug reports
- screenshots – API execution screenshots

## Tools

- Postman
- Swagger
- REST APIs
- JSON
